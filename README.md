# E-commerce Revenue Optimization Analysis
## Project Summary

This project analyzes historical e-commerce order data from 2018 to present to identify customer behaviors, product patterns, and trends that drive revenue growth.

The goal is to support better business decisions by transforming raw transactional data into structured SQL models and actionable insights.
## Business Problem

The business wants to improve profitability, but expense data is limited. This project focuses on revenue optimization by identifying which customers and products contribute most to long-term revenue.

The core question is: which customer segments and product behaviors drive the highest lifetime value, and how can these insights be used to increase revenue more efficiently?

## Dataset

The dataset contains historical e-commerce order data from 2018 to present.

It includes:
- Orders (transaction-level data)
- Customers
- Products / order items

Sensitive business information has been excluded or anonymized where necessary.
## Project Goals

- Clean and standardize raw order data
- Build structured SQL tables for analysis
- Create aggregated tables to improve query performance
- Analyze customer lifetime value and repeat purchasing behavior
- Identify product trends that drive revenue
- Build a dashboard to communicate insights
